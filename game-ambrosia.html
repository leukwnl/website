AMBROSIA

Browser-based physics sandbox showcasing WebGL particles and interaction.

Overview

AMBROSIA explores performant WebGL rendering and interactive particle simulations in the browser. The goal was to balance visual density with responsive input across devices.

My Role

  • Built the particle system and GPU-driven update loop
  • Implemented interaction tools and responsive UI
  • Profiling and optimization for mobile

Highlights

  • Instanced rendering for thousands of particles at 60 FPS
  • Explorable parameter space with presets
  • Cross-platform input handling
